# CebrianLab Visual Assets

This package contains initial visual assets for the CebrianLab website.

## Suggested folder in the website repository

static/images/
├── hero/
├── projects/
├── focus-icons/
└── backgrounds/

## Files included

- Hero images: homepage and research landing visuals.
- Project covers: one visual for each research project.
- Focus icons: one square icon for each of the 9 research focus areas.
- Backgrounds: abstract section backgrounds.
- AI prompt catalog: prompts for generating future AI images with a consistent visual identity.

## Recommended usage

- Use hero images in the Home and Research sections.
- Use project covers in the Projects / Research cards.
- Use focus icons in the Research Focus grid.
- Use backgrounds for Impact, Publications or Join Us sections.

All images are synthetic vector-style visuals created for the website draft. Replace them later with final AI-generated or professionally designed versions if needed.
